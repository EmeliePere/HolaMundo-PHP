# Hola Mundo en PHP

## Descripción
Este proyecto muestra un mensaje en pantalla utilizando PHP:  
**"Hola Mundo, Soy Emelie"**  
Además, incluye una imagen generada con inteligencia artificial.

## Prerrequisitos
Antes de ejecutar este proyecto, asegúrate de tener instalado:
- Sistema operativo Ubuntu/Linux
- Servidor web Apache o Nginx
- PHP (versión 7.4 o superior)

## Pasos para su ejecución
1. Clonar o descargar este repositorio.
2. Colocar el archivo `index.php` en la carpeta del servidor web (por ejemplo `/var/www/html`).
3. Abrir la terminal y ejecutar:
   ```bash
   php index.php
